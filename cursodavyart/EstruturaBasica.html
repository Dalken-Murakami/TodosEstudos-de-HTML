Tags estruturais básica em um arquivo HTML

HTML, HEAD, TITLE E BODY

Iremos usar todos os conceitos para contrução de tags que vimos anteriormente.


Começaremos pela tag raiz de todo e qualquer página html.

principais tags do html

  1. html : são tags que delimitam o documento html. É a raiz deste documento.
  2. head são tags que delimitam o cabelhaço
  3. title: correponde ao titulo da pagina
  4. body: responsavel por mostrar o conteudo da pagina
