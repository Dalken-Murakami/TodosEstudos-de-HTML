W3C

Fundado em 1994, por Tim berners- lee, o w3c é uma organização sem fins lucrativos composta por cerca de 400 membros cuaj missão é levar a www ao seu potencial máximo, desenvolvendo protocolos e diretrizes que garantam o crescimento a longo prazo da web.

  1. Web para todos
  2. Web em qualquer dispositivo.
  3. Web como uma base de conhecimento.
  4. Web confiável e segura.

Padrões WEB (Web Standards)

Os padrores web são as recomendações do w3c, as quais são destinadas a oreientar os desenvolvedores para o uso de boas práticas que tornam a web acessível para todos.
