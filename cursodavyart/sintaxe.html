Tags

Todo o documento HTML É FEITO A PARTIR DE ELEMTNSO DE MARCAÇÕES E ESTES SÃO FORMADOS ATRAVÉS DE TAGS. estas marcações definem o tipo de conteúdo e informam ao navegador como ele deve ser apresentado na página.

Tags são marcas específicas que delimitam o conteúdo, informado ao navegador sobre que tipo de informação se trata( um título, um parágrafo, uma lista). Toda tag contém um nome(nome da marca) e este também dá designação ao elemento HTML.

Todo elemento visual que você identifica em uma página web, desde um texto, uma foto ou um vídeo, são inseridos através de tags.

Elementos Nulos ( vazios ou void)

input / texto - ERRADO!
INPUT TEXT input errado.b
input / > - correto!
